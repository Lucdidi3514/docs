## Prérequis

<Check>Node.js 18+ installé</Check>
<Check>Compte Supabase créé</Check>
<Check>Git installé</Check>

## Installation

<Steps>
  <Step title="Cloner le projet">
```bash
    git clone https://github.com/votre-org/kitasso.git
    cd kitasso
    npm install
    </Step>
  <Step title="Configuration Supabase">
    Créez un fichier `.env.local` :
```bash
    NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
    SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```
<Tip>Récupérez ces clés dans votre dashboard Supabase → Project Settings → API</Tip>
  </Step>
  <Step title="Migrations de base de données">
```bash
    npm run db:push
    npm run db:seed  # (optionnel) données de test
```
  </Step>
  <Step title="Lancer le serveur">
```bash
    npm run dev
```
Ouvrez [http://localhost:3000](http://localhost:3000)
  </Step>
</Steps>
Vérification
